## 1. プロジェクト概要

水耕栽培完全素人MEGWINが、AI（Claude）の分析・予測・アドバイスを活用してプチトマトの水耕栽培に挑戦するリアルドキュメンタリーブログ。「ESP32で自動化」「Discord BotでAIが催促」「写真からAIが診断」など、他にない「AIが監督するスマート農場」を実現する。

### 1.1 コンセプト

「AIが監督する完全自動化スマート農場 × 素人ドキュメンタリー」

― MEGWINが作業し、Claude先生が分析・指示・記事を書く。人間×AIの共同栽培ブログ。

### 1.2 メインテーマ

- 素人でも始められる水耕栽培（初心者向け）
- 収益化を目指すリアルなストーリー（コスト記録・販売挑戦）
- AI×IoTの技術力を見せる（Gimmick Streamの技術を農業に応用）

### 1.3 目標

| フェーズ | 期間 | 目標 |
|---------|------|------|
| 前半: 夏の挑戦 | 4月〜8月 | 3株で比較実験、1株から1,000個収穫 |
| 後半: 通年栽培 | 9月〜翌年3月 | LED室内栽培で冬も枯らさない |
| 最終目標 | 継続 | プチトマト農家として収益化 |

## 3. AI自動更新システム

「更新が止まらないブログ」を実現する核心システム。

### 3.1 毎日のフロー

| 時刻 | アクション | 担当 |
|------|-----------|------|
| 朝10:00 | Discord Botが「今日のトマトの写真ちょうだい！」と催促 | AI自動 |
| -- | MEGWINがスマホで写真を撮ってDiscordに送信 | MEGWIN |
| -- | AIが「今日の調子は？」「気になることある？」と2〜3問質問 | AI自動 |
| -- | MEGWINが短く回答（「葉が黄色い」など） | MEGWIN |
| -- | 気温・天気・湿度をWeather APIから自動取得 | AI自動 |
| -- | ESP32センサーから水温・pH・EC値を取得 | IoT自動 |
| -- | 全データを統合して記事を自動生成 | AI自動 |
| -- | microCMSに下書き保存→Discordで確認通知 | AI自動 |
| -- | MEGWINが確認して公開ボタンを押す | MEGWIN |

※ 慣れたら確認なしで全自動公開に移行可能

### 3.2 記事構成

#### 前半: MEGWINパート（MEGWIN口調）

写真と、AIインタビューから生成したMEGWINらしい日記文。「今日もトマトがいい感じだぜ！」のようなノリ。

#### 後半: Claude先生のトマト診察室

- **現状分析**: 写真から葉の色・実の大きさ・成長ステージを判定
- **未来予測**: 気温データから「来週冷えるから注意」など先手アドバイス
- **大量生産アドバイス**: 「この段階で摘果すると収量UP」など具体的指示
- **次回の指示**: 「明日は液肥のpHチェックして」「葉の裏側の写真を送って」
- **緊急アラート**: 「葉が黄色い→窒素不足の可能性、すぐ対応を！」

### 3.3 データフロー（アーキテクチャ）

```
Discord Bot → 催促 → MEGWIN（写真+回答）
                            ↓
                      Supabase（全データ集約）
                        ↑         ↑
                Weather API    ESP32センサー
                (気象自動)     (水温/pH/EC)
                            ↓
                    Raspberry Pi
                  Claude Code起動
                 （Maxプラン内！）
                            ↓
                      記事自動生成
                 MEGWIN口調 + 先生分析
                            ↓
                    microCMS下書き投稿
                            ↓
                  Discord確認通知 → MEGWIN確認
                            ↓
                    megwin.com に公開
                    (Vercel + Next.js)
```

**ポイント: Claude Maxプラン内で完結。追加API課金なし。**

## 4. IoT自動化システム

Gimmick Streamで培ったESP32/ESP8266の技術を農業に応用する。

### 4.1 第1層: 完全自動（人間の手がいらない）

| 機能 | ハードウェア | 説明 |
|------|------------|------|
| 温度・湿度モニタリング | ESP32 + DHT22 | 常時計測、異常時Discordアラート |
| 水温・pH・EC値 | ESP32 + 各センサー | 液肥の状態を常時監視 |
| 水中ポンプ | ESP32 + リレー | 液肥の自動循環・入れ替え |
| LED照明制御 | ESP32 + LEDパネル | 波長・照射時間の自動制御 |
| エアポンプ | ESP32 + リレー | 溶存酸素供給の自動化 |
| 定点カメラ | ESP32-CAM / USBカメラ | 24時間タイムラプス撮影 |
| 気象データ | Weather API | 外気温・天気・降雨予報・日照時間 |

### 4.2 第2層: AIが指示→人間が実行

Claude先生が写真とデータを分析し、人間にしかできない作業を指示する。

- 「この脇芽を切ってください」（写真で場所を指定）
- 「摘果して実の数を減らしてください」
- 「定点カメラを設置しましょう」
- 「pHが下がってきたので液肥を調整して」
- 「来週冷えるので室内に移動して」（天気予報から先手アドバイス）

### 4.3 第3層: 記録・発信（ほぼ自動）

- 毎日の記事: Discordでの写真＋短い回答から自動生成
- コスト記録: レシート写真から自動集計（100円ショップ・Amazon購入）
- 成長データ: センサー+写真から全自動グラフ化

## 5. 比較実験計画

現在バーミキュライトで20cmまで育った苗を、これから水耕栽培へ移行するフェーズ。複数箇所で条件を変えて実験する。

### 5.1 実験条件の変数

- 場所: 室内 vs ベランダ vs 屋外 など
- エアポンプ: あり / なし
- 温度管理: 自然 vs ESP32制御
- 液肥濃度: 濃度を変えて比較
- LED補助光: あり / なし、波長の違い
- 設置場所の日照時間と日当たり具合

→ Claude先生が全データを比較分析し、「どの条件が最も収穫が多いか」をデータで導き出す。

## 8. まとめ

このプロジェクトは、3つの層が重なった唯一無二のコンテンツである。

### 第1層: 素人ドキュメンタリー

「素人が本当にプチトマト農家になれるのか？」というリアルなストーリー。コストも収益も全公開。

### 第2層: AIの力

「Claudeでここまでできるのか！」という技術的インパクト。写真分析、未来予測、自動記事生成、栽培アドバイス。

### 第3層: IoT自動化

Gimmick Streamの技術を農業に応用。ESP32センサー、ポンプ自動制御、LED波長制御。「真似できない領域」まで到達する。

この3つが掛け合わされることで、「ただのトマトブログ」ではなく、「AI時代の新しい農業の形」を示すコンテンツになる。
